Modules, Directives and Controllers

Modules

It is the container for the diferent parts of an aplication like the controllers

Directives

AngularJS directives are used to extend HTML. Directives are markers on HTML DOM element that tell AngularJS to attach a specified behavior to that HTML element.

Controllers

It controls the data of AngularJS applications
