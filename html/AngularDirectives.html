Angular Directives

Directives

  • ng-app

    Defines the part of the html in wich we can use angular functionalities

  • ng-model

    Defines the name of a variable that will store the data of an input

  • ng-bind

    Ouputs the data stored on the model directive

  • ng-init

    Initialices the data of the angular app

  • ng-repeat

    Repeats an HTML element

    The ng-repeat directive actually clones HTML elements once for each item in a collection. The ng-repeat directive can be used on an array of objects
