Angular expressions

Are defined using {{}}, they will output the data expresed inside.They can also be used as the bind directive, but can not be used with loops, exceptions or conditionals:

Examples

Data from a ng-init with two variables

Name: {{name}} | Sport: {{sport}}

Data from a ng-init JsObject

Name: {{person.firstName}} | Lastname: {{person.lastName}}

Data from a ng-init Array

Points: {{points[3]}}

Calculus expresions

x

Result: {{number1 * number2}}
